# Pyproject by Jackmanimation
## This is the placeholder for the pyproject project
This is a personal project which stores various snippets of python that I am working on.
I use a couple of github actions (see below). And I will add to this as I find elements to add.

## Snippets
* Discord
* Excel
* Slack
* Spotify
* Trello

### Discord
This is a simple discord bot which sits inthe server and responds accordingly.

### Excel
This is an example of writing to and reading from MS Excel Spreadsheets and workbooks.

### Slack
This is the beginning of a slack bot. My aim here is to make a bot which will broadcast to a umber of channels on demand.

## Spotify

## Trello

## GitHub actions
### PyLint
I have pylint double check the code when it is checked in and I act on the result to keep this green.

### pages-guild-deployment
This action builds these pages when they are committed.
